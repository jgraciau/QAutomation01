# Documentación de pruebas

Este directorio contiene los planes de prueba y documentación del framework de automatización.

- `docs/test-plan-opencart.md`: Plan de pruebas del flujo de checkout de OpenCart.

- Mantener los planes actualizados junto con los casos de prueba.
